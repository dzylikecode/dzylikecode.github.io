# Margin Note

## 主要流程

应该注重的是**知识的关系**, 而不是**知识的记录**

注重 content 的关系, 思维导图的层次不要太深, 否则拆开来

转化到 thought, 然后转为 markdown

书名作为工作区, 完成一个章节, 就将这个章节的内容提出来
