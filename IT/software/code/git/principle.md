# 基本原理

![](assets/2022-09-09-23-29-37.png)

## add

- `git add`

## remove

- `git rm -r --cached`

  - `--cached`: remove from index
  - `-r`: recursive

## commit

- 修改上一次的提交信息

  ```bash
  git commit --amend
  ```
